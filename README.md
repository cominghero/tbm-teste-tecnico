# Technical test TBM Home Care

This repo is my submission for the Full Stack Developer (Laravel + React) position at TBM Home Care.

The test is split into three exercises, one folder each:

- [`ex1-code-review/`](./ex1-code-review): review of the `AtendimentoController` code that came with the test. I list the problems I found (tagged critical, high or medium), explain the order I'd fix them in, and actually rewrite the top 3 critical ones.
- [`ex2-feature/`](./ex2-feature): the `POST /api/v1/checkin` endpoint built from scratch. Uses Sanctum, enforces multi-tenant isolation, and ships with feature tests.
- [`ex3-raciocinio/`](./ex3-raciocinio): short written answers to the four reasoning questions.

Each folder has its own README with the decisions I made there. This one is the executive summary and lives at the top.
